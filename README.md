# Movie Recommendation System

A content-based movie recommendation system that suggests movies similar to a user's selection by analyzing movie metadata such as genres, keywords, cast, and director. The application is built with Python and Streamlit, while movie posters are fetched dynamically using the TMDB API.

# Project Overview

Finding movies that match a user's interests can be challenging given the large number of available titles. This project implements a content-based recommendation system that suggests similar movies by comparing their metadata, including genres, keywords, cast, and director.

The application uses TF-IDF Vectorization and Cosine Similarity to measure the similarity between movies and recommends the top matches. Movie posters are retrieved dynamically using the TMDB API and displayed through a Streamlit web application.

---

# Features

- Content-based movie recommendations
- Interactive Streamlit web application
- Top 10 similar movie suggestions
- Movie posters fetched using the TMDB API
- Recommendation generation using TF-IDF Vectorization
- Fast retrieval using a precomputed similarity matrix

# Recommendation Approach

This project implements a Content-Based Filtering recommendation system.

Instead of relying on user ratings or viewing history, recommendations are generated by comparing the content of each movie.

The following features are used:

- Genres
- Keywords
- Cast
- Director

These features are combined into a single text representation called `tags`.

TF-IDF Vectorization converts the textual data into numerical feature vectors, while Cosine Similarity measures the similarity between movies to identify the most relevant recommendations.

---

# Workflow

1. Load and preprocess the movie dataset.
2. Combine genres, keywords, cast, and director into a single `tags` feature.
3. Convert the textual data into TF-IDF vectors.
4. Compute cosine similarity between all movies.
5. Store the similarity matrix for efficient retrieval.
6. Accept a movie selected by the user.
7. Retrieve the most similar movies.
8. Fetch movie posters using the TMDB API.
9. Display the recommendations in the Streamlit application.

# Installation

### Clone the repository

```bash
git clone https://github.com/MoazzamFarooqui/Recommendation-System.git
```

### Navigate to the project directory

```bash
cd Recommendation-System
```

### Install the required dependencies

```bash
pip install -r requirements.txt
```

### Configure the TMDB API key

Create a `.env` file in the project root.

```env
API_KEY=YOUR_TMDB_API_KEY
```

### Run the application

```bash
streamlit run app.py
```

# Usage

1. Launch the Streamlit application.
2. Select a movie from the dropdown list.
3. Click the Recommend button.
4. View the top recommended movies along with their posters.
